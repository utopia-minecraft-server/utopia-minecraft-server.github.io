## 📝 Редактирование контента

### Добавление новостей

Отредактируйте файл `news.json`:

```json
{
  "news": [
    {
      "date": "YYYY-MM-DD",
      "title": "Заголовок новости",
      "content": "Содержание новости"
    }
  ]
}
```

### Добавление информации о сервере

Отредактируйте файл `server-content.json`:

```json
{
  "buildings": [
    {
      "name": "Название постройки",
      "description": "Описание"
    }
  ],
  "clans": [
    {
      "name": "Название клана",
      "leader": "Имя лидера",
      "description": "Описание клана"
    }
  ]
}
```

## 🚀 Расширение функциональности

Сайт легко расширяется благодаря модульной структуре:

1. Добавьте новый раздел в `index.html`
2. Создайте новый JSON файл для данных
3. Добавьте функцию загрузки в `script.js`
4. Добавьте стили в `styles.css`
